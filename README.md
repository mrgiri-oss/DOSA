# 🌐 Delhi Odia Students Association (DOSA) Website

A modern, responsive, and animation-driven website built for the **Delhi Odia Students Association (DOSA)**.
This project focuses on smooth user experience, elegant UI design, and engaging scroll-based interactions.

---

## ✨ Features

* 🎯 Smooth scrolling experience using **Lenis**
* 🎬 Scroll-based animations powered by **GSAP & ScrollTrigger**
* 🧩 Interactive UI elements (sliders, hover effects, modals)
* 📱 Fully responsive design (mobile + desktop)
* 📌 Sticky sections and structured layouts
* 🎨 Clean and minimal design with a strong visual hierarchy

---

## 🛠️ Tech Stack

* **HTML5**
* **Tailwind CSS**
* **JavaScript (Vanilla JS)**
* **GSAP (GreenSock Animation Platform)**
* **ScrollTrigger**
* **Lenis (Smooth Scroll Library)**

## 🚀 Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/mrgiri-oss/dosa-website.git
   ```

2. Navigate to the project folder:

   ```bash
   cd dosa-website
   ```

3. Open `index.html` in your browser.
